## 개발 도구 설치

Simaple을 개발하기 위해서는 다음 도구들이 설치되어 있어야 합니다.

- [Node.js](https://nodejs.org/) >= 20
- [Poetry](https://python-poetry.org/docs/#installation)

## 개발 서버 실행

```bash
poetry run poe dev
```

## webui 실행

```bash
cd webui
yarn install # 의존성 설치가 필요할 때만 실행
yarn start
```

webui는 기본적으로 https://localhost:5173 에서 실행됩니다.

개발 서버와 webui가 성공적으로 실행되었다면 다음 단계로 진행할 수 있습니다.
