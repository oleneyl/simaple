# Simaple Web UI

## Getting Started

### Prerequisites

- Node.js >= 18.0.0
- yarn

### Installation

```bash
yarn install
```

### Dev Server

웹앱 개발 서버를 실행합니다.

```bash
yarn dev
```

로컬에서 빌드된 simaple wheel을 Pyodide가 http로 접근할 수 있도록 서버를 추가로 실행해야 합니다.

```bash
yarn dev:wasm
```

### Build

```bash
yarn build
```
